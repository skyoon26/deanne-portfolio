What I've Built

Here are the projects I've worked on so far. Please stay tuned for more!

Well Coffee Inventory

As part of a four-person team, I contributed to developing a responsive online inventory tracking system for our LaunchCode capstone project. This platform allows managers and employees to log, review, and reorder items, streamlining inventory management for small businesses. Designed for a faux coffee shop, it features low-stock alerts, purchase order creation, a database search function, and integrates real-time Amazon data via Rapid API, providing an efficient solution for inventory tracking on desktop and mobile devices.

Technologies: React, Bootstrap, Java Spring Boot, MySQL, Axios, Real-Time Amazon Data (Rapid API)

Employee Management Application

This Employee Management App is my first full-stack application, designed to allow users to efficiently manage employee data. The app supports CRUD (Create, Read, Update, Delete) operations, enabling users to add new employees, view existing records, update information, and remove employees as needed.

Technologies: React, Java Spring Boot, MySQL

Weather Application

I created this project to gain experience connecting to public APIs by utilizing OpenWeather's Weather API. While following an online tutorial, I added personal features to make the app more unique and user-friendly. Notable improvements include a toggle switch for switching between metric and imperial units, as well as the ability to submit searches using either the enter key or a button click, enhancing usability.

Technologies: HTML, CSS, JavaScript

TroveTrack

TroveTrack is an enhanced version of my LaunchCode capstone project, rebuilt from the ground up to improve both functionality and user experience. My goal was to expand on the original project by gaining hands-on experience with features I wasn’t directly responsible for during the program. I redesigned the UI to create a more modern, intuitive interface with streamlined navigation, while also overhauling the backend to incorporate robust authentication and authorization using Spring Security. Additionally, I maintained integration with RapidAPI's Amazon product data, allowing users to link inventory items via Amazon identification numbers for more detailed tracking.

Technologies: React, Bootstrap, Java, Spring Boot, Spring Security, MySQL, Axios, Real-Time Amazon Data (Rapid API)
