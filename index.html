What I've Built

Here are the projects I've worked on so far. Please stay tuned for more!

Well Coffee Inventory

As part of a four-person team, I contributed to developing a responsive online inventory tracking system for our LaunchCode capstone project. This platform allows managers and employees to log, review, and reorder items, streamlining inventory management for small businesses. Designed for a faux coffee shop, it features low-stock alerts, purchase order creation, a database search function, and integrates real-time Amazon data via Rapid API, providing an efficient solution for inventory tracking on desktop and mobile devices.

Technologies: React, Bootstrap, Java Spring Boot, MySQL, Axios, Real-Time Amazon Data (Rapid API)

Employee Management Application

This Employee Management App is my first full-stack application, designed to allow users to efficiently manage employee data. The app supports CRUD (Create, Read, Update, Delete) operations, enabling users to add new employees, view existing records, update information, and remove employees as needed.

Technologies: React, Java Spring Boot, MySQL

Weather Application

I developed this weather app to gain experience connecting to public APIs, utilizing OpenWeather's Weather API. The app features a simple interface that allows users to search for weather information by city. Notable enhancements include a toggle switch for switching between Celsius and Fahrenheit, and the ability to submit searches using either the enter key or a button click for improved usability.

Technologies: HTML, CSS, JavaScript
